BSGS 算法 ( $\texttt{Baby Step Giant Step}$ )

基本概念

这里由题目 P3846 [TJOI2007] 可爱的质数/【模板】BSGS 引入算法。

给定一个质数 $p$，以及一个整数 $b$，一个整数 $n$，现在要求你计算一个最小的非负整数 $l$，满足 $b^l\equiv n(\bmod\,p)$。

BSGS 算法用于求解高次同余方程，即给定整数 $a,b,p$，其中 $a$ 与 $p$ 互质，求满足 $a^x\equiv b(\bmod\,p)$ 的最小非负整数解 $x$。

由扩展欧拉定理知，$a^x\equiv a^{x\bmod\varphi(p)+\varphi(p)}(\bmod\,p)$，由于 $a$ 与 $p$ 互质，因此 $a^x\equiv a^{x\bmod\varphi(p)}(\bmod\,p)$。

于是，式子 $a^x\equiv b(\bmod\,p)$ 就可以变为 $a^{x\bmod\varphi(p)}\equiv b(\bmod\,p)$，由于 $\varphi(p)<p$，因此就可以遍历 $x\in[0,p]$ 就能够找到最小的正整数 $x$。

此时使用暴力枚举 $x$ 的方法的时间复杂度为 $O(p)$，因此考虑优化算法。

设 $x=im-j$，其中 $m=\Big\lceil\sqrt{p}\Big\rceil,i\in[1,m],j\in[0,m-1]$，即 :

  • 当 $i=1$ 时，$j=m-1$，则 $x=im-j=1\times m-(m-1)=1$。
  • 当 $i=m$ 时，$j=0$，则 $x=im-j=m\times m-0=p$。

此时对于 $x=0$ 的情况进行特判即可。

  1. 先枚举 $j$，将 $(ba^j,j)$ 放入哈希表中，如果键值 $ba^j$ 重复了，则使用更大的 $j$ 代替旧的，因为在在式子 $x=im-j$ 中，更大的 $j$ 可以使 $x$ 的值更小。

  2. 接着枚举 $i$，计算 $(a^m)^i$，并在哈希表中查找对应的键值，找到第一个即可结束，因为此时的 $(a^m)^i$ 最小，可以使 $x$ 的值更小。

枚举 $j,i$ 的次数都是 $\sqrt{p}$ 次的,因此时间复杂度为 $O(\sqrt{p})$。

算法的本质是分块思想 :

将 $x\in[1,p]$ 分成 $\sqrt{p}$ 块，每块长度都为 $\sqrt{p}$。

在枚举 $j$ 时，

类比为婴儿的一小步 ( $\texttt{Baby Step}$ )。

而在枚举 $i$ 时，

类比为巨人的一大步 ( $\texttt{Giant Step}$ )。

为什么要求 $a$ 与 $p$ 互质?

  1. 扩展欧拉定理的成立需要满足 $a$ 与 $p$ 互质。

  2. 由 $(a^m)^i\equiv ba^j(\bmod\,p)$ 推回 $a^{im-j}\equiv b(\bmod\,p)$ 需要同余式两边同时乘上 $a^j$，所以必须有 $a^j$ 与 $p$ 互质，同时 $a$ 与 $p$ 互质。

    在同余式两侧同时乘上 $a^j$ 就要保证 $a^j$ 为 $(a^m)^i$ 和 $ba^j$ 在模 $p$ 的意义下的乘法逆元，即需要保证 $a^j$ 与 $p$ 互质。

    例如

    $15\equiv3(\bmod\,4)\implies5\equiv1(\bmod\,4)$

    $15\equiv3(\bmod\,6)\implies5\not\equiv1(\bmod\,4)$

    同时这也是乘法逆元的作用的体现。

扩展 BSGS 算法 ( $\texttt{Extended Baby Step Giant Step}$ )

基本概念

对于 BSGS 算法，必须要满足的条件是 $a$ 与 $p$ 互质，否则扩展欧拉定理和乘法逆元就不再成立。而扩展 BSGS 算法就可以不需要这个条件。

因此进行分类讨论，对于同余方程 $a^x\equiv b(\bmod\,p)$ :

  • 当 $a$ 与 $p$ 互质时，可以直接使用 BSGS 算法求解。
  • 当 $a$ 与 $p$ 不互质时，可以考虑使两者互质。

高斯消元法 ( $\texttt{Gaussian Elimination}$ )

高斯消元法可以求解线性方程组。

对于系数矩阵 $A$，

  1. 首先枚举主元 $x_i$，对于一个主元 $x_i$，找到一行主元系数不为 $0$ 的行。
  2. 使用初等行变换 $1$，将改行与第一行交换。
  3. 使用初等行变换 $2$，将主元系数变为 $1$。
  4. 使用初等行变换 $3$，将其他行的主元系数变为 $0$。
  5. 反复执行前 $4$ 步操作，直到枚举完成所有主元。

最终就可以得到一个上三角矩阵，可以看到左上角就类似于一个三角形，再从下到上回带求解。

例如上面的矩阵中，可以很轻易的从第 $n$ 行得出 $x_n=b’_n$，再带入第 $n-1$ 行得出 $x_{n-1}=b’_{n-1}-a’_{n-1,n}\times x_n$。

例如下面的矩阵，枚举的主元为 $x_1$。

按照高斯消元法就可以交换第 $1$ 行和第 $2$ 行。

随后使用变换 $2$，将主元的系数改为 $1$，即全部系数乘以 $0.5$。

接着将其余行的主元系数改为 $0$，即将第 $3$ 行减去第 $1$ 行的 $2$ 倍。

然后枚举主元为 $x_2$，并将第二行的主元系数改为 $1$。

同样需要将其余行的主元系数改为 $0$。

走后还需要将主元枚举为 $x_3$，并将最后一行的主元系数改为 $1$。

此时不难看出 $x_3=3$，并回带至上一行，使 $b’_2$ 减去 $a’_{2,3}\times x_3$ 变为 $2$。

这样也不难看出此时 $x_2=2$，并且可以接着回带至上一行，使 $b’_1$ 减去 $a’_{1,2}\times x_2+a’_{1,3}\times x_3$ 变为 $1$。

最后求解出 $x_1=1$，整一个求解完成。

此时方程组不一定有唯一解，还可能存在无解或无穷多解的情况。

  1. 唯一解，最终枚举完所有主元 $x_i$。
  2. 无解，存在一行 $k$ 的主元系数 $a’_{k,k}$ 只能为 $0$ 且 $b’_k\neq0$，此时不论 $x_k$ 是任何实数，都无法满足该行的线性方程。
  3. 无穷多解，存在一行 $k$ 的主元素 $a’_{k,k}$ 只能为 $0$ 且 $b’_k=0$，此时不论 $x_k$ 是任何实数，都可以满足该行的线性方程。

高斯-约旦消元法 ( $\texttt{Gauss-Jordan Elimination}$ )

高斯-约旦消元法相比较于高斯消元法更加简单，高斯-约旦消元法会先将矩阵消元成为主对角矩阵，再使用等式右侧的数除以主元系数。

矩阵求逆

基本概念

对于矩阵 $A$，若存在矩阵 $B$，使得 $A\times B=B\times A=I$，则称矩阵 $B$ 为矩阵 $A$ 的逆矩阵，记作 $A^{-1}$ ( $I$ 为单位矩阵 )。

给出一个 $n\times n$ 的矩阵 $A$，求解其逆矩阵 $A^{-1}$ 的方法。

  1. 构造一个 $n\times 2n$ 的矩阵 $(A,I)$，将矩阵 $A$ 与矩阵 $I$ 拼接起来。
  2. 用高斯-约旦消元法将其简化为 $(I,A^{-1})$，即可得到 $A$ 的逆矩阵 $A^{-1}$。

证明 :

可以看做 $n$ 组 $n$ 元线性方程组的矩阵形式，设下面的方程为 $A\times X=I$

此时每一个被分开的矩阵 $X$ 的列都是矩阵 $I$ 中的一列，都有其各自的矩阵，最后依次拼接即可。

因此可以得到

矩阵 $A$ 变为了单位矩阵 $I$，由于两侧是相等的，因此单位矩阵 $I$ 就变为了矩阵 $A$ 的逆矩阵 $A^{-1}$。

当左部出现了全 $0$ 行时，说明此时不可能构造出单位矩阵，则矩阵 $A$ 是不可逆的。
