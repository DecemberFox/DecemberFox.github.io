排列组合 ( $\texttt{Permutation and Combination}$ )

基本概念

  • 排列 : 从 $n$ 个元素中，按照不同顺序取出 $m(m\le n)$ 个元素并排为一列叫做排列，其中不同的方法的总数叫做排列数 ( 记作 $P^m_n$ 或 $A^m_n$ )。

  • 组合 : 从 $n$ 个元素中，取出 $m(m\le n)$ 个元素所组成的集合叫做组合，其中不同的方法总数叫做组合数 ( 记作 $C^m_n$ )。

排列与组合的区别 : 选取出的数若在意顺序则为排列，若不在意顺序则为组合。

公式法求排列/组合数

  • 排列 :

    从 $n$ 个元素中按照不同顺序取出 $m$ 个元素排为一列叫做排列，排列首先需要从 $n$ 个元素中取出第一个元素，于是第一次选取时就有 $n$ 种选取方法。

    接下来选取第二个元素时，由于第一次选取的元素已经不能选取了，序列中仅剩下 $n-1$ 个元素，因此就有 $n-1$ 种选取方法。

    不断进行选取，最后选取完所有 $m$ 个元素，根据乘法原理，每次选取都是一个步骤，则总共有 $n\times(n-1)\times(n-2)\times\cdots\times(n-m+1)$ 种方法。

    仔细观察上式，不难发现就是从 $(n-m+1)$ 到 $n$ 的连乘积，就相当于 $\frac{n!}{(n-m)!}$。

    因此得到求排列数的公式

  • 组合 :

    根据排列的定义，在排列中会出现 $\{a,b,c\}$ 和 $\{b,a,c\}$ 等等元素组成成分相同，但是顺序不同的情况，但是在组合中就会去掉这类重复的情况。

    假设已经完成了对一个数列求排列数，那么对于任意一种组合方式 $\{a,b,c,\cdots,z\}$，其一定有若干种重复的排列方式。不难想到相对于组合方式，排列方式会多出全排列数种情况，即是原排列的 $\frac{1}{m!}$，随后再带入排列公式内，就可以得到求组合的公式

    可以得到，对于 $C^m_n$ :

    • 若第 $i$ 个数选择取出，则余下数列的组合数为 $C^m_{n-1}$。

    • 若第 $i$ 个数选择不取出，则余下数列的组合数为 $C^{m-1}_{n-1}$。

基本概念

给定整数 $n,m,p$，保证 $p$ 为质数，则有 :

为证明卢卡斯定理，先证明下面两个引理 :

  • 引理 $1$ : $C^x_p\equiv0(\bmod\,p)$ 且 $0<x<p$

因 $C^x_p=\frac{p!}{x!(p-x)!}=\frac{p(p-1)!}{x(x-1)!(p-x)!}=\frac{p}{x}\times\frac{(p-1)!}{(x-1)![(p-1)-(x-1)]!}=\frac{p}{x}C^{x-1}_{p-1}$

故 $C^x_p\equiv p\cdot x^{-1}C^{x-1}_{p-1}(\bmod\,p)$

由于 $0<x<p$，因此 $x$ 关于 $p$ 的乘法逆元 $x^{-1}$ 一定存在且为整数，由于 $C^{x-1}_{p-1}$ 也为整数，因此式子 $x^{-1}C^{x-1}_{p-1}$ 同样为整数，因此 $p\cdot x^{-1}C^{x-1}_{p-1}$ 为 $p$ 的倍数，即

引理 $1$ 证毕。

  • 引理 $2$ : $(1+x)^p\equiv1+x^p(\bmod\,p)$

由二项式定理得 $(1+x)^p=\sum^p_{i=0}C^i_p x^i$，当式子中 $i\in[1,p-1]$ 时，由引理 $1$ 得，$C^i_p\equiv0(\bmod\,p)$，因此这些项都不存在了，仅剩下 $C^0_p\times x^0+C^p_p\times x^p=1+x^p$

引理 $2$ 证毕。

证明 : $C^m_n\equiv C^{\lfloor m\div p\rfloor}_{\lfloor n\div p\rfloor}C^{m\bmod p}_{n\bmod p}(\bmod\,p)$

令 $n=ap+b,m=cp+d$

在 $[1]$ 式中 $x^m$ 的系数为 $C^m_n$。

在 $[2]$ 式中 $x^m=x^{cp+d}=x^{cp}\cdot x^d$，对应的系数为 $C^c_aC^d_b$。

隔板法 ( $\texttt{Stars and Bars}$ )

正整数和的组数

若 $\forall x_i\ge1$ 且 $n$ 为正整数，求 $x_1+x_2+\cdots+x_k=n$ 的整数解的组数。

这道题如果按照常规的方法思考就比较难以想，因此可以转化问题，假设有 $n$ 个完全相同的球，现在需要将其分为 $k$ 组，且保证每一组的球的个数大于等于 $1$，共有多少种方法。

再次转化问题，将 $n$ 个球排成一排，则会有 $n-1$ 个空位，于是现在考虑将 $k-1$ 块板子插入空位中，使得一排球恰好分成 $k$ 组。此时就将为题转换为了一个简单的组合数问题，即将 $k-1$ 块板子插入 $n-1$ 个空位的方案数。

明显地，答案为 $C^{k-1}_{n-1}$。

例如在 $4$ 个球中间插入 $2$ 块板子，共有 $3$ 种方法 :

非负整数和的组数

若 $\forall x_i\ge0$ 且 $n$ 为非负整数，求 $x_1+x_2+\cdots+x_k=n$ 的整数解的组数。

此时可以令 $y_i=x_i+1$，则 $y_i\ge1$，那么 $y_1+y_2+\cdots+y_k=n+k$，则问题变为了正整数和的组数，可以使用上述方法求解，即 $C^{k-1}_{n+k-1}$。

不同下界整数和的组数

若 $x_i\ge a_i\ge0$ 且 $n$ 为大于 $\sum a_i$ 的非负整数，求 $x_1+x_2+\cdots+x_k=n$ 的整数解的组数。

令 $y_i=x_i-a_i+1$，则 $y_i\ge1$，那么 $y_1+y_2+\cdots+y_k=n-\sum a_i+k$，则问题变为了正整数和的组数，可以使用上述方法求解，即 $C^{k-1}_{n-\sum a_i+k-1}$。

题目

P1450 [HAOI2008] 硬币购物

共有 $4$ 种硬币。面值分别为 $c_1,c_2,c_3,c_4$。

某人去商店买东西，去了 $n$ 次，对于每次购买，他带了 $d_i$ 枚 $i$ 种硬币，想购买 $s$ 的价值的东西。请问每次有多少种付款方法。

其中保证 $1 \leq c_i, d_i, s \leq 10^5$，$1 \leq n \leq 1000$。

容易考虑使用多重背包求解，但是时间复杂度为 $O(4sn)$，显然无法通过本题。

首先将问题抽象化，即求解 $\sum^4_{i=1}c_ix_i=s$，并要求满足 $x_i\le d_i$ 的非负整数解的个数。直接求解满足 $x_i\le d_i$ 条件下的交集较为困难，考虑先求不受限制条件下的全集，然后减去 $x_i\ge d_i+1$ 条件下的补集的并。

先使用完全背包预处理在硬币无限制时的方案数 $f$，此时 $f_s$ 就为全集的大小。

然后考虑使用容斥原理求解不满足条件的情况的集合的并，即不满足一种硬币情况的集合的方案数减去不满足两种硬币情况的方案数再加上不满足三种硬币情况的方案数并以此类推。

如果此时假设只有第 $k$ 种硬币超额，就需要减去 $f_{s-c_k(d_k+1)}$，可以这样理解，即凑齐 $s-c_k(d_k+1)$ 金额后就可以使用 $d_k+1$ 次金额为 $c_k$ 的硬币满足金额 $s$，自然方案数就为 $f_{s-c_k(d_k+1)}$。

但是为什么在 $f_{s-c_k(d_k+1)}$ 中，并不使用 $d_k+2$ 或 $d_k+3$，原因是在初始化过程中的状态转移方程 $f_j=\sum^4_{i=1}f_{j-c_i}$ 中已经将 $c_k(d_k+1)$，$c_k(d_k+2)$ 以及后续的 $d_k+n$ 的过程中累加合并了，因此就无需去依次累加了。

由于本题仅有 $4$ 种硬币，想要完全枚举完是很快的。

由于需要求解容斥原理需要执行 $n$ 次，因此总的时间复杂度为 $O(4\times2^4\times n)$，即 $O(64n)$，是非常快的。
